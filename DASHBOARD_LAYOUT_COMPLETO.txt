## 🎯 COMO USAR

### 1. Usar o novo layout:
```blade
@extends('layouts.dashboard')

@section('title','Minha Página')
@section('page-title','Título da Página')
@section('page-subtitle','Descrição da página')

@section('content')
  {{-- Conteúdo aqui --}}
@endsection
```

### 2. Grid de 2 colunas:
```blade
<div class="grid-2">
  <div class="card">Coluna 1</div>
  <div class="card">Coluna 2</div>
</div>
```

### 3. Card full-width:
```blade
<div class="card span-2">Ocupa 2 colunas</div>
```

### 4. Botões:
```blade
<button class="btn ghost">Secundário</button>
<button class="btn primary">Primário</button>
```
