## 🎯 MODOS DISPONÍVEIS

### 1. Full (padrão) - Ícone + "Abrir no Maps"
```blade
<x-map-link :href="optional($cliente)->maps_url" mode="full" />
```
Resultado: 📍 Abrir no Maps

### 2. Compact - Ícone + "Mapa"
```blade
<x-map-link :href="optional($cliente)->maps_url" mode="compact" size="sm" />
```
Resultado: 📍 Mapa

### 3. Icon Only - Só ícone (acessível com sr-only)
```blade
<x-map-link :href="optional($cliente)->maps_url" mode="icon" size="sm" />
```
Resultado: 📍 (tooltip: "Abrir no Google Maps")

---

## 📋 PARÂMETROS

| Parâmetro | Tipo | Padrão | Descrição |
|-----------|------|--------|-----------|
| `href` | string | null | URL do Google Maps |
| `label` | string | "Abrir no Maps" | Texto do botão |
| `mode` | string | "full" | 'full', 'compact', ou 'icon' |
| `size` | string | "md" | 'sm' (compacto) ou 'md' (normal) |
| `title` | string | null | Tooltip customizado |

---

## 🎨 EXEMPLOS DE USO

### Em tabelas (compacto):
```blade
<x-map-link :href="optional($p->cliente)->maps_url" mode="icon" size="sm" />
```

### Em detalhes (completo):
```blade
<x-map-link :href="optional($pedido->cliente)->maps_url" mode="full" label="Ver rota no Maps" />
```

### Em cards (intermediário):
```blade
<x-map-link :href="optional($cliente)->maps_url" mode="compact" size="sm" />
```
