```php
    Route::resource('cupons', CuponsController::class);
});
```

### 2. Verificar/Adaptar Model Cupom

**Importante:** O controller assume um Model `App\Models\Cupom` com campos:
- `codigo` (string)
- `descricao` (string|null)
- `tipo` ("percent"|"valor")
- `valor` (decimal)
- `minimo_pedido` (decimal|null)
- `uso_maximo` (int|null)
- `validade_inicio` (date|null)
- `validade_fim` (date|null)
- `ativo` (bool)

### 3. Recomendado: Adicionar casts no Model

```php
protected $casts = [
    'ativo' => 'boolean',
    'validade_inicio' => 'datetime',
    'validade_fim' => 'datetime',
];
```
