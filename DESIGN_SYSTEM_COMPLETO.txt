## 🎯 COMO USAR

### **1. Page Header:**
```blade
<x-page-header title="Produtos">
  <a href="{{ route('products.create') }}" class="btn btn-primary">+ Novo Produto</a>
</x-page-header>
```

### **2. Stat Card:**
```blade
<x-stat-card label="Cupons Ativos" value="{{ $stats['ativos'] ?? 0 }}" />
```

### **3. Search:**
```blade
<div class="card" style="padding:16px; margin-bottom:16px">
  <x-search name="q" placeholder="Buscar produtos..." />
</div>
```

### **4. Grid:**
```blade
<div class="grid grid-3">
  @foreach($items as $item)
    <div class="card" style="padding:18px">...</div>
  @endforeach
</div>
```

### **5. Tabela:**
```blade
<div class="card" style="padding:8px">
  <table class="table">
    <thead>...</thead>
    <tbody>...</tbody>
  </table>
</div>
```

### **6. Badges:**
```blade
<x-badge type="success">Ativo</x-badge>
<x-badge type="danger">Inativo</x-badge>
<x-badge type="warning">Pendente</x-badge>
```

## 🎨 CUSTOMIZAÇÃO

### **Paleta de Cores:**
```css
--bg: #F7F7F7;          /* Fundo geral */
--surface: #FFFFFF;     /* Cards/superfícies */
--text: #1f2937;        /* Texto principal */
--muted: #6b7280;       /* Texto secundário */
--primary: #ff7a1a;     /* Cor principal (laranja) */
--primary-600: #fb6a00; /* Hover da cor principal */
--success: #22c55e;     /* Sucesso */
--warning: #f59e0b;     /* Aviso */
--danger: #ef4444;      /* Perigo */
--ring: #e5e7eb;        /* Bordas */
```

