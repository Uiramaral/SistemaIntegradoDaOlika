## 🎯 COMO USAR

### Menu Lateral

**Desktop:**
- Sidebar fixa à esquerda
- Sempre visível
- Item ativo destacado

**Mobile:**
- Clique ☰ para abrir
- Clique fora para fechar
- Pressione ESC para fechar
- Clique em link para navegar e fechar
- Scroll bloqueado quando aberto

### Visual

**Cores:**
- Background: #fafafa
- Cards: #fff
- Linhas: #ececec
- Brand: #ff7a18 (laranja vibrante)

**Bordas:**
- Cards: 18px
- Inputs/Botões: 12px

## 💡 DICAS

### Se uma página não carregar:
Verifique se tem `@section('content')`:
```blade
@extends('layouts.dashboard')
@section('title','Título')
@section('page-title','Título')
@section('page-subtitle','Subtítulo')
@section('content')
  {{-- Conteúdo aqui --}}
@endsection
```

### Personalizar cores:
Edite `public/css/styles.css`:
```css
:root{
  --brand:#ff7a18; /* Laranja */
  --line:#ececec;  /* Bordas */
  --radius:18px;  /* Arredondamento */
}
```
