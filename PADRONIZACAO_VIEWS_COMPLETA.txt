## 🎨 COMPONENTES DISPONÍVEIS

### **Toolbar** (cabeçalho)
```html
<div class="toolbar">
  <div class="grow">
    <h1 class="page-title">Título</h1>
    <p class="page-sub">Subtítulo</p>
  </div>
  <div class="actions">
    <a class="btn ghost">Ação 1</a>
    <a class="btn primary">Ação 2</a>
  </div>
</div>
```

### **Card com Search**
```html
<div class="card">
  <div class="input-search">
    <svg class="icon" width="18" height="18">...</svg>
    <input class="inp" placeholder="Buscar...">
  </div>
</div>
```

### **Badges**
```html
<span class="badge green">Ativo</span>
<span class="badge orange">Em preparo</span>
<span class="badge gray">Pendente</span>
```

### **Avatar**
```html
<div class="avatar">JS</div>
```

### **Pills** (botões de ação)
```html
<a class="pill" href="#">Ação</a>
```

### **Form Grid** (PDV)
```html
<div class="field-row">
  <div class="field" style="grid-column: span 6;">
    <label class="lbl">Label</label>
    <input class="inp" placeholder="...">
  </div>
</div>
```
